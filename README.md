# 📚 Page Turners Bookstore

A simple bookstore website built using HTML, CSS, JavaScript, and Bootstrap 5.

## Features

- Responsive navigation bar
- Home page
- About page
- Books page
- Events page
- Gallery page
- Contact page
- FAQ page

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
